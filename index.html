Add the EST UI widget to your web page:

  1. Make sure you have jquery, or add it to yor website: 
    
<script type="text/javascript" src="https://ajax.googleapis.com/ajax/libs/jquery/1.11.3/jquery.min.js"></script>
  2. Grab the source files.
  3. Add the following to your page's <head>: 
    
<link rel="stylesheet" type="text/css" href="est-ui-widget/style.css">
<script type="text/javascript" src="est-ui-widget/index.js"></script>
<script>
    $(document).ready(function ($) {
         $("#est-example-widget-container").est_widgetize({
               api_key: "Your API key" // get it from http://api.trade.gov/
         });
    });
</script>
  4. Add <div id="est-example-widget-container"></div> to wherever you wish the widget to appear.
  5. Note: you can pick another id for the div #est-example-widget-container, always` remember to change the id in the script when changing the div id.
